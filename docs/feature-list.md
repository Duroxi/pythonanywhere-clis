# pythonanywhere-cli 项目

## 项目目的

实现本地 Web 项目的全自动化部署到 PythonAnywhere。

用户通过 AI 生成前端页面（或其他 Web 项目）后，本工具负责将本地项目一键变为线上网站，返回一个可访问的 PythonAnywhere 地址。整个过程对用户透明，由 CLI 命令或 AI Skill 驱动，覆盖从账号准备到网站上线的全部环节。

核心价值：**本地项目 → 线上网站，一步到位。**

## 实现状态

**已完成**: 47 个命令，454 测试通过，91% 覆盖度。

## 实现路径

为了达成上述目的，需要实现以下全部原子化能力，使 AI Agent 和用户都能自由编排部署流程。
