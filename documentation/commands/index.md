## Quick Reference

### Account Management
- `pa init` - Configure account
- `pa register` - Register new account
- `pa account list` - List accounts
- `pa account switch` - Switch account
- `pa account token` - Fetch API token

### Deployment
- `pa deploy <dir>` - One-click deploy
- `pa deploy <dir> --dry-run` - Preview deploy

### File Management
- `pa files ls` - List files
- `pa files upload` - Upload files
- `pa files download` - Download files

### Console Management
- `pa console list` - List consoles
- `pa console create` - Create console
- `pa console send <id> <cmd>` - Send command

### Web App Management
- `pa webapp create` - Create web app
- `pa webapp config` - Configure web app
- `pa webapp reload` - Reload web app
